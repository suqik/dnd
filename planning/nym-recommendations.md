# Nym 推荐记录

这份文件只记录给 Nym 的玩法、升级、法术和战术建议，不作为角色事实记录。Nym 的个人状态、物品和 XP 以 [Nym 个人角色卡](../characters/nym-copperstar.md) 为准；队伍任务与剧情进展以 [KEN 小队与战役记录](../campaign/ken-party.md) 为准。

最后更新：2026-07-09

## 当前判断

- 角色路线不需要改：继续走预言法师。
- 小队是三人：Nym 法师、蛋蛋牧师、卡尔战士。
- 小队短板：人数少、前排只有卡尔、缺少专业斥候和稳定群体处理手段。
- Nym 的定位：调查与奥术解题优先，战斗中承担控场、补伤害、保命和关键骰干预。

## 3 级升级建议

升到 3 级时，先按本桌采用的规则确认预言法师学者类能力如何触发：

- 若本桌按“3 级取得 2 环法术位时同时触发预言学者”处理，则法术书大概率新增：2 个任意法师法术 + 3 个预言系法师法术。
- 若本桌不在这次升级触发该条款，则先按普通法师升级处理：新增 2 个任意法师法术。

### 推荐方案 A：本次升级获得额外预言法术

普通升级的 2 个法术：

- 蛛网术 Web：核心控场，帮助三人小队把敌人分割、定住、让卡尔和远程攻击更容易命中。
- 粉碎音波 Shatter：补足三人队缺少群体伤害的问题，尤其适合敌人密集、需要快速清杂兵时。

预言系额外法术优先：

- 侦测思想 Detect Thoughts：调查、审讯、社交场景价值高。
- 定位物品 Locate Object：找黑水晶、导师遗物、被藏物资时很实用。
- 看破隐形 See Invisibility：防隐形敌人和异常魔法威胁。

备选：若本桌将占卜 Augury 纳入可选法术范围，它也很贴合预言主题，但实际价值取决于桌内信息反馈方式。

### 推荐方案 B：只有普通新增 2 个法术

如果接下来继续野外、亡灵、黑潮、战斗较多：

- 蛛网术 Web
- 粉碎音波 Shatter

如果接下来明显回城调查、审讯、追查导师记号：

- 蛛网术 Web
- 侦测思想 Detect Thoughts

## 3 级常备准备法术建议

3 级准备法术数会变成 6 个。护盾术 Shield 来自魔法启蒙，仍然额外可用，不占准备名额。

偏野外 / 亡灵 / 战斗：

- 法师护甲 Mage Armor
- 魔法飞弹 Magic Missile
- 蛛网术 Web
- 粉碎音波 Shatter
- 羽落术 Feather Fall
- 睡眠术 Sleep 或侦测思想 Detect Thoughts

偏城镇 / 调查 / 神殿和执法线：

- 法师护甲 Mage Armor
- 魔法飞弹 Magic Missile
- 蛛网术 Web
- 侦测思想 Detect Thoughts
- 羽落术 Feather Fall
- 睡眠术 Sleep、隐形术 Invisibility 或看破隐形 See Invisibility

备注：侦测魔法、通晓语言、鉴定术、寻获魔宠是法术书里的仪式法术时，通常不需要准备也能仪式施放；只有时间压力很大时才考虑准备。

## 魔法启蒙替换建议

当前不急着替换。

- 护盾术 Shield：保留。它是低级法师最重要的保命反应。
- 次级幻影 Minor Illusion：保留。调查、误导、潜入和创造机会都很强。
- 传讯术 Message：可留。若实际使用率很低，升级时可按本桌采用的替换规则考虑心灵裂片 Mind Sliver。

心灵裂片更偏战斗辅助，能帮后续控制法术压豁免；传讯术更偏潜入、分头行动和队伍沟通。

## 短期战术建议

- 战斗前能预判危险时，优先开法师护甲。
- 卡尔顶住敌人后，Nym 不要急着追伤害，优先用蛛网术或睡眠术改变敌人行动。
- 敌人聚集且不方便控场时，用粉碎音波。
- 对强敌或关键目标，预兆低骰优先给对方的豁免；预兆高骰优先给卡尔的关键攻击、队友死亡豁免或关键调查检定。
- 猫头鹰魔宠主要负责侦查、Help、望风和递送触碰法术，不要让它长期停在敌人旁边。
- 三人队没有余裕硬换血，Nym 的第一原则是保持距离和维持专注。

## 4 级前后建议

4 级最稳仍然是属性提升：

- 智力 +1、体质 +1，让 INT 到 18，CON 到 16。

如果战斗中经常被打断专注，再考虑：

- 战地施法者 War Caster
- 坚韧：体质 Resilient (Con)

如果剧情调查和读心沟通越来越重要，再考虑：

- 心灵感应 Telepathic
- 敏锐头脑 Keen Mind
- 观察者 Observant

## 后续桌规问题

- 预言学者在 3 级是否给 3 个额外预言系法术。
- 2024 PHB 之外的法术是否开放，例如心灵裂片。
- 户外使用蛛网术时，锚点、地面铺网和燃烧蛛网如何裁定。
- 当前僵尸、黑潮亡灵是否免疫睡眠、魅惑、恐惧等状态。
- 城镇里是否有卷轴、法术书或法师 NPC，可以让 Nym 抄录法术。
